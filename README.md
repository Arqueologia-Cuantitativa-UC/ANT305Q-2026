## 📂 Estructura del Repositorio
Si deseas explorar los archivos fuente o descargar los datos, el repositorio está organizado de la siguiente manera:

* `index.Rmd` / `index.html`: Archivos raíz que generan el índice del curso.
* `Clases/`: Carpeta que contiene los scripts (`.Rmd`) y las páginas generadas (`.html`) de cada sesión lectiva.
* `datos/`: Bases de datos (archivos `.xlsx` o `.csv`) necesarias para ejecutar los ejercicios prácticos en RStudio.
